## Viewing HTML Files

Clone the repo and open any `index.html` directly in a browser — no build step required. All HTML files are self-contained with a shared stylesheet at `assets/style.css`.

```bash
git clone https://github.com/dshipe/30a.git
cd 30a
open posts/index.html          # Overview
open posts/seaside/index.html  # Seaside
```

## About 30A

Scenic Highway 30A runs along Florida's South Walton coast, connecting a string of architecturally distinct beach communities on the Emerald Coast. Each town has its own personality — together they make up one of the most distinctive stretches of coastline in the American South, known for sugar-white quartz sand and emerald-green Gulf water.
